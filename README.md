# PetCare Landing

Статичний лендинг на чистих HTML/CSS/JS.

## Структура

- `landing/index.html`
- `landing/styles.css`
- `landing/script.js`
- `landing/assets/` (директорія для локальних ассетів)

## Як запустити

```bash
cd landing
python -m http.server 8000
```

Відкрийте у браузері: `http://localhost:8000`
